# 外婆生前交代的事

如同徽州传统女性一样，外婆同样希望死后有一个好的旌表，所以生前一直交待，死后要为她写一篇好的祭文。
如同古徽州的一座座牌坊一样，
无不在讴歌表颂这些伟大的女性。

老家老一辈的传统，
自己的子孙写祭文，
而且能读得通，
才是子孙有望，
也代表逝者的光荣。

# 不是所有人家的祭文都是自己写的

用我们村长的话说：一辈子看到由自己子孙亲自写祭文的，30年前看过一回，那是村里地主家的儿子，为其父亲补的葬礼（其父于解放后因反革命罪被枪毙，80年代子孙重设葬礼）。
第二回就是占才女的葬礼。

世风日下，已经没有人愿意为自己的长辈亲自写祭文了。
但我还是坚持自己持笔，同时需要符合徽州古韵。
道士先生说：
我有一个祭文的模板，送你抄一下吧？

我很果断的摇头拒绝了，
以我的心眼和文采，
怎么会用人家的模板呢，
当年的文科状元，
诗三百倒背如流，
自有骨气。

# 葬礼上我亲自用曹门话大声朗颂祭文

最后我哭成泪人
读完后
死劲的以头跄地
要是那一刻
撞死自己
这也是值的

婶婶和占才治安表舅很快把我抱了起来
在悲痛中的祭文
回了外婆的一生

# 每个外婆都是伟大的

而我的外婆为我付出更多
外婆的儿子、孙子不孝顺
跪在边上听我的祭文
想必心中自然会有内疚
即使不知内疚
也自然会有街坊邻居知道
一切付与秋风

# 我的外婆是最伟大的

外婆你在我的心中早已树起了伟大的牌坊 你也在世人的眼间
树起了一个牌坊
诉说一个普通的女性
如何成为伟大
如何塑造了外甥
如何回报了社会

# 来年，

还会为您作祭
秋风有知
当入我梦

# 归来兮

且祝且悼

青山作证！

附记：葬礼上，外婆儿子作祭时，道士问：是否要帮你读祭文。舅舅说：不用，我能读。
在老家，如果自己不能读祭文，就要给道士付一个红包。
道士写祭文也一样要付红包。
当然，大多数人家都是给道士红包解决，没人去写祭文，也读不顺。
结果舅舅读了第一句，就读不下去了，结果只好让道士代祭。
这是一个侧记。
